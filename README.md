# Temperature-Prediction-Model
## Overview
This project predicts temperature using environmental data like humidity and pressure.

## Models Used
- Linear Regression
- Random Forest Regressor

## Results
- Linear Regression MSE: 16.47
- Random Forest MSE: 3.53

## Key Learnings
- Data preprocessing
- Handling missing values
- Model training and evaluation
- Model comparison

## Tools & Libraries
- Python
- Pandas
- Scikit-learn
- Matplotlib
